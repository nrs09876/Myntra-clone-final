# Myntra-clone-final
Developed a responsive Myntra Clone using React.js, HTML, CSS, and Redux with features like dynamic product listing, cart management, and modern UI design.


Myntra Clone is a fully responsive e-commerce web application developed using React.js, HTML, CSS, and Redux. The main objective of this project is to replicate a real-world online shopping experience where users can browse products, view product details, add items to the cart, and manage their shopping flow smoothly.

Redux is used for global state management to efficiently handle cart operations and product data. The application follows modern UI/UX design principles and responsive layout techniques, ensuring seamless performance across desktop, tablet, and mobile devices.
